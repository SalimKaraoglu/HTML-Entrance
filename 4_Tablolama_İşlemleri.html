Sayfa üzerinde bir takım verileri tablo formatında göstermek için sunuklan element "table" elementidir. Temel olarak üç şey vardır. "tr" "td" "th"

TD(table-data) => kolon bilgisini TH(table-header) => başlık bilgisini TR(table-row) => ise satır bilgisini temsil eder.

Bazı Özellikleri:

  • Width => Tablonun genişliğini verir
  • Border => Tablonun içi ve dış çerçeveleri için bir boyut verir
  • Cellspacıng => Hücrelerin birbirlerine olan uzaklıklarını verir
  • Cellpeding => Hücre içerisindeki değerlerin dış çerçeveye olan uzaklıklarını verir
  • Colspan => Hücrelerin merge edilmesini sağlayan öz nitelik(attibute) vermiş olduğumuz iki değerin iki sütunmuş gibi davranmasını sağlar
  • Rowspan => Satırların merge edilmesini sağlar
